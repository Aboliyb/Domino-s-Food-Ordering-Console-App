Project Overview:

This project is a console-based food ordering system inspired by Dominoâ€™s Pizza, developed using Core Java.
It allows users to select pizzas, choose sizes, and view order details through a command-line interface.

Features :

Choose between Veg and Non-Veg Pizza
Select pizza size (Small, Medium, Large)
Display order details
Calculate total bill
Interactive console menu

Technologies Used :

Java (Core Java)
OOP Concepts (Classes, Objects, Abtraction)
Control Statements (if-else, switch)

How to Run the Project:

Clone the repository:
git clone https://github.com/Aboliyb/Domino-s-Food-Ordering-Console-App.git
Open the project in any IDE (Eclipse / IntelliJ)
Compile the program:
javac Main.java
Run the program:
java Main
